git init: Tao repo tren may
git clone: Tải từ mạng về trong máy
git pull: Đồng bộ từ mạng về trong máy

git add   or   git add . : Thêm file vào repo trên máy
git commit -m "nội dung message" : Kết thúc add và đề lại message
git push: Đẩy tha đổi từ repo trên máy lên mạng

git log  or   git status: Xem trạng thái file như thế nào
git log --decorate --graph --oneline
